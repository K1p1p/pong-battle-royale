# pong-battle-royale
Cool little online pong with support for multiple players. Base game provided by Willian Oliveira

---

# Requirements
- NodeJS
- TS-Node

# Install
```sh
npm install ts-node
```

# Running
## Server
```sh
ts-node ./src/server/server.ts
```

## Client
```
Open src/client/index.html in browser
```
